## Breeze 统计相关包概览

Breeze 中与统计、概率分布相关的核心包是：

```scala
import breeze.stats._
import breeze.stats.distributions._
```

### 核心内容

- **概率分布（distributions）**

	- 离散分布：Bernoulli, Binomial, Poisson
    - 连续分布：Gaussian, Uniform, Exponential, Gamma, Beta

- **统计量（stats）**

	- 均值、方差、协方差
    - `mean` / `variance` / `meanAndVariance`

## Breeze 常用统计分布

### 公共导入

```scala
import breeze.stats._
import breeze.stats.distributions._
import breeze.stats.meanAndVariance

implicit val rand: RandBasis = RandBasis.mt0
// 或：import Rand.VariableSeed._
// 或：import Rand.FixedSeed._
````

---

### 1. 伯努利分布 Bernoulli

```scala
val bern = new Bernoulli(0.3)
val samples = bern.sample(100)

bern.probabilityOf(1)
mean(samples)
```

### 2. 二项分布 Binomial

```scala
val binom = new Binomial(10, 0.5)
val samples = binom.sample(100)

binom.probabilityOf(3)
mean(samples)
```

### 3. 泊松分布 Poisson

```scala
val pois = new Poisson(2.0)
val samples = pois.sample(100)

samples.map(i => pois.probabilityOf(i))

val doublePois = for (x <- pois) yield x.toDouble
meanAndVariance(doublePois.samples.take(1000))
```

### 4. 正态分布 Gaussian

```scala
val gauss = new Gaussian(0.0, 1.0)
val samples = gauss.sample(100)

gauss.pdf(0.0)
mean(samples)
variance(samples)
```

### 5. 均匀分布 Uniform

```scala
val uni = new Uniform(0.0, 1.0)
val samples = uni.sample(100)

uni.pdf(0.5)
mean(samples)
```

### 6. 指数分布 Exponential

```scala
val exp = new Exponential(2.0)
val samples = exp.sample(100)

exp.pdf(1.0)
mean(samples)
```

### 7. Gamma 分布

```scala
val gamma = new Gamma(2.0, 1.0)
val samples = gamma.sample(100)

gamma.pdf(1.0)
mean(samples)
```

### 8. Beta 分布

```scala
val beta = new Beta(2.0, 5.0)
val samples = beta.sample(100)

beta.pdf(0.3)
mean(samples)
```

## 常用统计函数

```scala
mean(samples)
variance(samples)
meanAndVariance(samples)
```
