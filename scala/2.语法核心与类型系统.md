## 变量

scala的变量分为两类，可变与不可变，分别用val和var来声明
```scala
val a:Int = 1 // Immutable
var b:Int = 1 // mutable

b = 2 // OK，mutated b
scala> a = 2
		// ^
		//error: reassignment to val
```

定义变量时不必声明类型，scala会根据变量的赋值自动推断类型
```scala
val a = 1
// scala> a
// val res0: Int = 1
```

scala的变量是静态类型的
```scala
var b:Int = 1
b = "abc"
//^
//  error: type mismatch;
//	found   : String("abc")
//	required: Int
```

同一个变量在同一局部作用域内只能声明一次，在内层作用域中重复声明外部作用域的变量则会触发遮蔽（shadowing）
